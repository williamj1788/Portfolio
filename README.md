## How to install

> To view or edit this project, follow these instructions
>
> ``` terminal
> git clone https://github.com/williamj1788/Portfolio
> cd Portfolio
> npm install
> ```
>
> The project should open up in a new tab
>
> If it doesn't, you can view it on localhost 3000
